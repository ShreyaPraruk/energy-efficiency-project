# ⚡ Energy Efficiency & Waste Analysis Project

## 📊 Project Overview

This project focuses on analyzing energy efficiency across different plants and machines using a real-world industrial dataset. The goal is to identify inefficient machines, analyze energy consumption patterns, and build predictive and analytical models.

---

## 🧩 Tools & Technologies Used

* 📈 **Excel** – Data cleaning, preprocessing, and basic analysis
* 🗄️ **SQL (MySQL)** – Data aggregation, querying, and efficiency analysis
* 🐍 **Python (Colab)** – Regression modeling & anomaly detection
* 📊 **Power BI** – Interactive dashboards and visual insights

---

## 📁 Project Structure

```
Energy-Efficiency-Project/
│
├── data/        → Dataset (CSV format)
├── excel/       → Excel analysis file
├── sql/         → SQL queries
├── python/      → Python notebook (Colab)
├── powerbi/     → Power BI dashboard
```

---

## 🔍 Key Analysis

### ✅ SQL Analysis

* Monthly energy metrics by plant
* Total Energy Consumption & Production Units
* Top 10 inefficient machines
* Efficiency flag using statistical threshold (mean + standard deviation)

---

### ✅ Python Analysis

* Regression models to predict **Energy Consumption**
* Compared multiple models using RMSE
* Anomaly detection using Z-score / Isolation Forest

---

### ✅ Power BI Dashboard

* Energy consumption trends over time
* Energy per unit comparison by plant
* Identification of worst-performing machines
* Interactive filters for deeper insights

---

## 📈 Key Insights

* Identified machines with high energy usage per unit
* Detected abnormal spikes in energy consumption
* Provided data-driven insights to improve operational efficiency

---

## 🎯 Conclusion

This project demonstrates an end-to-end data analytics workflow combining multiple tools to solve real-world industrial problems and improve energy efficiency.
